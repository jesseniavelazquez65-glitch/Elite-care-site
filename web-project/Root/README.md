# Elite Care Co.

A responsive, accessible multi-page website built with HTML, CSS, and JavaScript.

## Features
- Multi-page navigation
- Accessible form with ARIA support
- EmailJS integration (real email delivery)
- Lighthouse-optimized
- Mobile-first responsive design

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- EmailJS
